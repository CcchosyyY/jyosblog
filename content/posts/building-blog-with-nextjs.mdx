# Next.js로 블로그 만들기

이 글에서는 Next.js 14를 사용하여 블로그를 만드는 과정을 정리합니다.

## 사용한 기술 스택

- **Next.js 14**: React 기반 풀스택 프레임워크
- **TypeScript**: 타입 안정성
- **Tailwind CSS**: 유틸리티 퍼스트 CSS
- **MDX**: 마크다운 + JSX

## 프로젝트 구조

```
MyBlog/
├── app/           # App Router 페이지
├── components/    # 재사용 컴포넌트
├── content/       # MDX 블로그 글
├── lib/           # 유틸리티 함수
└── styles/        # 스타일 파일
```

## 주요 기능

1. **정적 사이트 생성 (SSG)**: 빠른 페이지 로딩
2. **MDX 지원**: 마크다운으로 쉽게 글 작성
3. **다크모드**: 시스템 설정에 따른 자동 전환
4. **반응형 디자인**: 모바일/데스크톱 지원
